# 闭环演示视频脚本（建议 6-10 分钟）

## 1. 开场（30秒）
- 展示系统总览：后台大厅“管网运维闭环”模块入口

## 2. 工单全生命周期（2分钟）
- 新建工单 -> 分配 -> 开始 -> 暂停 -> 继续 -> 送审 -> 通过 -> 关闭
- 演示 `rejected` 与 `reopen`

## 3. 管网-楼宇联动（1.5分钟）
- 工单详情展示受影响楼宇/房间与避让要求
- 手动调整影响范围并展示调整日志

## 4. 巡检与维修联动（1.5分钟）
- 上传巡检记录（含水压/水质/照片/位置/判定）
- 一键转维修并展示自动预填

## 5. 热水泵控制（1分钟）
- 批量控制、进度条、倒计时、前后状态对比
- 成功后地图刷新

## 6. 三维联动与看板（1分钟）
- 看板热力点聚合，点击跳转工单详情
- 地图定位高亮：管线/楼宇/房间不同颜色

## 7. 收尾（30秒）
- 展示统计看板与测试报告/迁移对比材料目录
